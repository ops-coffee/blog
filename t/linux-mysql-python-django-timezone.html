各系统或服务查询时区方法汇总

Linux系统查看时区

# cat /etc/timezone
Asia/Shanghai

or

# date +'%z'
+0800

MySQL [(ops-coffee.cn)]> show VARIABLES like '%time_zone%';

| Variable_name    | Value  |
+------------------+--------+
| system_time_zone | CST    |
| time_zone        | SYSTEM |
+------------------+--------+
