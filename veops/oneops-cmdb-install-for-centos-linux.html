超详细纯净Linux系统安装部署VECMDB

VECMDB是一个功能完善灵活强大的开源的CMDB系统，详细介绍看这里：灵活多变功能强大的CMDB，在开始之前你只需要准备一台主机，并安装好linux操作系统即可

以下操作环境基于CentOS7

安装Docker

1.更新系统包

yum update -y

2.安装必要的依赖

yum install -y yum-utils

3.添加 Docker 的官方存储库

yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

4.安装 Docker 引擎

yum install -y docker-ce docker-ce-cli containerd.io

5.启动 Docker 并将其设置为开机启动

systemctl start docker
systemctl enable docker

安装 Docker Compose v2

1.安装 Docker Compose 插件

yum install -y docker-compose-plugin

2.验证 Docker Compose 是否安装成功

docker compose version

安装GIT

yum install git

启动项目

1.创建项目目录

mkdir /data
cd /data

2.拉项目代码并启动

git clone https://github.com/veops/cmdb.git
cd cmdb/
docker compose up -d

访问项目

浏览器打开，输入http://ip:8000访问，默认账号密码分别为：
